# Comins Contract v1.2 Sensitive Data Standard Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Establish Contract v1.2, a framework-neutral sensitive-data standard, and a self-hosted prevention gate in `comins-governance` without performing remote writes.

**Architecture:** Governance owns the normative policy and a zero-production-dependency Node.js reference scanner. The scanner exposes pure detector functions plus worktree, file, staged, push, history, and package modes; local hooks, required CI, and checksum-pinned Gitleaks form independent enforcement layers.

**Tech Stack:** Markdown, Node.js 24 ESM, Node test runner, Git hooks, GitHub Actions, Gitleaks 8.30.1, npm lockfile with no third-party packages.

## Global Constraints

- Keep `comins-governance`, `comins-table`, `comins-grid-layout`, and `comins-sortable` as independent Git repositories, CI pipelines, and commit units.
- Limit this plan to future prevention; do not rewrite Git history, delete published npm metadata, unpublish packages, or request historical cache removal.
- Do not add an npm production or development dependency.
- Never commit an actual personal name, personal email, local account path, credential, API key, private key, token, or secret-derived fingerprint as a fixture or allowlist value.
- Allow only public repository handles, GitHub noreply identities, documented service addresses, explicit placeholders, and neutral paths.
- Failure output contains only a rule ID, non-sensitive repository-relative path or ordinal redacted label, and line number.
- Treat scanner errors, unreadable input, malformed Git ranges, symlink traversal, archive traversal, unsupported archive entries, and checksum mismatch as failures.
- Keep workflow permissions at `contents: read`.
- Do not push, open or update a PR, change GitHub settings, publish, or perform another external write without a separate explicit request.

```markdown
# Comins Sensitive Data Standard

This document is the conformance reference for Comins Contract v1.2.

## Protected Data

- Personal identifiers: personal names, personal email addresses, identifying home paths, phone numbers, Korean resident-registration-number formats, account UI, unrelated production or support data.
- Credentials: provider API keys and tokens, bearer credentials, JWTs, passwords, client secrets, private keys, credential-bearing URLs, `.env`, `.npmrc`, `.netrc`, service-account, credential, and key-container files.
- Public exceptions: repository handles, GitHub noreply identities, documented service addresses, explicit placeholders, and neutral paths such as `<repo-root>`.

## Required Modes

- `worktree`: repository-owned tracked and untracked files plus ignored sensitive filenames.
- `file`: one explicitly selected outbound file before external attachment or copy.
- `staged`: exact staged filenames and blob contents.
- `push`: every new commit and tag selected by pre-push input.
- `history`: selected local and remote commit, tag, message, metadata, filename, and text-blob history.
- `package`: the exact `.tgz` produced for artifact upload or npm staging.

## Output Contract

A finding contains only rule ID, path label, and line number. The scanner must not print the matched value. If a path contains protected data, use a per-run ordinal label such as `[redacted-path:1]`; do not hash the original path.

## Failure Contract

Exit `0` only when the requested scope is completely scanned and clean. Exit `2` for findings. Exit `1` for scanner, input, file, Git, archive, or configuration errors. Every nonzero result blocks the calling hook, CI job, or release job.

## Allowlist Contract

The version-controlled allowlist contains only `ruleId`, repository-relative `path`, and `reason`. Every entry requires maintainer review. Values, personal identifiers, token fingerprints, and secret-derived hashes are forbidden.

## Required Layers

Use repository-owned staged and push hooks, required full-history CI, GitHub Secret Scanning and Push Protection, checksum-pinned Gitleaks with redaction, and exact package-archive scanning. Hooks are installed explicitly per clone and are never activated by an install lifecycle script.
```

const textRules = [
  ["private-key", /-----BEGIN (?:RSA |DSA |EC |OPENSSH |PGP )?PRIVATE KEY-----/g],
  ["github-token", /\b(?:gh[pousr]_[A-Za-z0-9]{20,}|github_pat_[A-Za-z0-9_]{20,})\b/g],
  ["npm-token", /\bnpm_[A-Za-z0-9]{20,}\b/g],
  ["aws-access-key", /\b(?:AKIA|ASIA)[A-Z0-9]{16}\b/g],
  ["openai-api-key", /\bsk-(?:proj-|svcacct-)?[A-Za-z0-9_-]{20,}\b/g],
  ["google-api-key", /\bAIza[0-9A-Za-z_-]{35}\b/g],
  ["slack-token", /\bxox[baprs]-[0-9A-Za-z-]{10,}\b/g],
  ["stripe-live-key", /\b(?:sk|rk)_live_[0-9A-Za-z]{16,}\b/g],
  ["jwt", /\beyJ[A-Za-z0-9_-]{8,}\.[A-Za-z0-9_-]{8,}\.[A-Za-z0-9_-]{8,}\b/g],
  ["credential-url", /\b[a-z][a-z0-9+.-]*:\/\/[^\s/:@]+:[^\s/@]+@/gi],
  ["auth-header", /\bAuthorization\s*[:=]\s*["']?(?:Bearer|Basic)\s+[A-Za-z0-9+/_.=-]{12,}/gi],
  ["npm-auth-config", /(?:^|\n)\s*(?:\/\/[^\s:]+\/)?_authToken\s*=\s*[^\s${<][^\s]*/g],
  ["personal-home-path", /\/(?:Users|home)\/(?!<(?:user|repo-root)>\/|runner\/|home\/|user\/)[^\s/'"`]+\//g],
  ["korean-rrn", /\b\d{6}-[1-4]\d{6}\b/g],
  ["korean-phone", /\b01[016789]-\d{3,4}-\d{4}\b/g],
];
const emailPattern = /\b[A-Z0-9._%+-]+@[A-Z0-9.-]+\.[A-Z]{2,}\b/gi;
const secretAssignment = /\b(?:api[_-]?key|access[_-]?token|auth[_-]?token|client[_-]?secret|password|passwd|secret)\b\s*[:=]\s*(?:"([^"\n]{8,})"|'([^'\n]{8,})'|([^\s,;}]{8,}))/gi;
const sensitiveFilename = /(?:^|\/)(?:\.env(?:\..+)?|\.npmrc|\.netrc|\.pypirc|id_(?:rsa|dsa|ecdsa|ed25519)(?:\..+)?|[^/]*(?:credential|service[-_]?account|private[-_]?key)[^/]*|[^/]+\.(?:pem|key|p12|pfx|jks|keystore))$/i;
const allowedEmail = (value) =>
  /@users\.noreply\.github\.com$/i.test(value) ||
  /^noreply@github\.com$/i.test(value) ||
  /@(example\.(?:com|org|net)|localhost)$/i.test(value);
const allowedAssignment = (value) =>
  /^(?:example|dummy|test|changeme|replace[-_ ]?me|your[-_ ])/i.test(value) ||
  /\$\{|process\.env|import\.meta\.env|<[^>]+>/.test(value);
const lineAt = (text, index) => text.slice(0, index).split("\n").length;
const finding = (ruleId, path, line) => ({ ruleId, path, line });

function protectedPath(path, markers = []) {
  emailPattern.lastIndex = 0;
  const normalizedPath = path.normalize("NFKC").toLocaleLowerCase("en-US");
  return emailPattern.test(path) ||
    /\/(?:Users|home)\//i.test(path) ||
    markers.some((marker) => normalizedPath.includes(marker.normalize("NFKC").toLocaleLowerCase("en-US")));
}

export function createPathLabeler(markers = []) {
  const labels = new Map();
  return (path) => {
    if (!protectedPath(path, markers)) return path;
    if (!labels.has(path)) labels.set(path, `[redacted-path:${labels.size + 1}]`);
    return labels.get(path);
  };
}

export function scanText(path, text, labelPath = (value) => value) {
  const findings = [];
  for (const [ruleId, pattern] of textRules) {
    pattern.lastIndex = 0;
    for (const match of text.matchAll(pattern)) findings.push(finding(ruleId, labelPath(path), lineAt(text, match.index ?? 0)));
  }
  emailPattern.lastIndex = 0;
  for (const match of text.matchAll(emailPattern)) {
    if (!allowedEmail(match[0])) findings.push(finding("non-placeholder-email", labelPath(path), lineAt(text, match.index ?? 0)));
  }
  secretAssignment.lastIndex = 0;
  for (const match of text.matchAll(secretAssignment)) {
    const value = match[1] ?? match[2] ?? match[3] ?? "";
    if (!allowedAssignment(value)) findings.push(finding("secret-assignment", labelPath(path), lineAt(text, match.index ?? 0)));
  }
  return [...new Map(findings.map((item) => [`${item.ruleId}|${item.path}|${item.line}`, item])).values()];
}

export function scanFilename(path, labelPath = createPathLabeler(), markers = []) {
  const label = labelPath(path);
  const findings = [];
  if (sensitiveFilename.test(path)) findings.push(finding("sensitive-filename", label, null));
  if (protectedPath(path, markers)) findings.push(finding("sensitive-path", label, null));
  return findings;
}

export function scanIdentity(path, name, email, owners, markers, labelPath = (value) => value) {
  const normalized = name.trim().toLocaleLowerCase("en-US");
  const noreplyHandle = /^(?:\d+\+)?([^@]+)@users\.noreply\.github\.com$/i
    .exec(email.trim())?.[1]?.toLocaleLowerCase("en-US");
  const findings = scanText(path, email, labelPath).filter((item) => item.ruleId === "non-placeholder-email");
  const publicName = owners.has(normalized) ||
    normalized === noreplyHandle ||
    /^(?:github-actions|dependabot)(?:\[bot\])?$/.test(normalized) ||
    normalized === "github";
  if (normalized && !publicName) {
    findings.push(finding("non-public-identity", labelPath(path), null));
  }
  if (markers.some((marker) => name.normalize("NFKC").toLowerCase().includes(marker.normalize("NFKC").toLowerCase()))) {
    findings.push(finding("personal-identity", labelPath(path), null));
  }
  return findings;
}

export function scanLocalMarkers(path, text, markers, labelPath = (value) => value) {
  const normalizedText = text.normalize("NFKC").toLocaleLowerCase("en-US");
  const findings = [];
  for (const marker of markers) {
    const normalizedMarker = marker.normalize("NFKC").toLocaleLowerCase("en-US");
    if (!normalizedMarker) continue;
    const index = normalizedText.indexOf(normalizedMarker);
    if (index >= 0) findings.push(finding("local-identity-marker", labelPath(path), lineAt(normalizedText, index)));
  }
  return findings;
}

export function applyAllowlist(findings, allowlist) {
  if (allowlist?.version !== 1 || !Array.isArray(allowlist.entries)) throw new Error("Invalid security allowlist");
  for (const entry of allowlist.entries) {
    const keys = Object.keys(entry ?? {}).sort().join(",");
    if (!entry?.ruleId || !entry?.path || !entry?.reason || keys !== "path,reason,ruleId") {
      throw new Error("Every allowlist entry requires ruleId, path, and reason only");
    }
  }
  return findings.filter((item) => !allowlist.entries.some((entry) => entry.ruleId === item.ruleId && entry.path === item.path));
}

export function formatFindings(findings) {
  return findings.map(({ ruleId, path, line }) => `${ruleId}: ${path}${line ? `:${line}` : ""}`).join("\n");
}

  lstatSync,
  mkdtempSync,

import { fileURLToPath } from "node:url";

  applyAllowlist,
  createPathLabeler,
  formatFindings,
  scanFilename,
  scanIdentity,
  scanLocalMarkers,
  scanText,

const scriptsRoot = dirname(fileURLToPath(import.meta.url));
const root = dirname(scriptsRoot);
const allowlist = JSON.parse(readFileSync(join(root, "security-allowlist.json"), "utf8"));
const markerFile = join(root, ".git", "comins-sensitive-markers");
const markers = existsSync(markerFile)
  ? readFileSync(markerFile, "utf8").split(/\r?\n/).map((line) => line.trim()).filter((line) => line && !line.startsWith("#"))
  : [];
const labelPath = createPathLabeler(markers);
const decoder = new TextDecoder("utf-8", { fatal: true });
const zeroObject = /^0{40,64}$/;
const revisionToken = /^[A-Za-z0-9._/@~^:+-]+$/;
const binaryExtensions = /\.(?:woff2?|ttf|otf|png|jpe?g|gif|webp|ico|pdf|zip|gz|tgz)$/i;
const excludedPath = /^(?:node_modules|dist|coverage|test-results|playwright-report|reports\/artifacts|\.worktrees)(?:\/|$)/;

function git(args, binary = false) {
  return execFileSync("git", args, {
    cwd: root,
    encoding: binary ? null : "utf8",
    maxBuffer: 64 * 1024 * 1024,
    stdio: ["ignore", "pipe", "pipe"],
  });
}

function normalizePath(input) {
  const normalized = input.split(sep).join("/").replace(/^\.\//, "");
  if (!normalized || normalized.startsWith("../") || isAbsolute(normalized)) {
    throw new Error("Path outside repository");
  }
  return normalized;
}

function ownerHandles() {
  const remote = git(["remote", "get-url", "origin"]).trim();
  const owner = /github\.com[/:]([^/]+)\//i.exec(remote)?.[1]?.toLocaleLowerCase("en-US");
  if (!owner) throw new Error("Origin owner unavailable");
  return new Set([owner, "github", "github-actions", "github-actions[bot]", "dependabot", "dependabot[bot]"]);
}

function decodeText(buffer, path) {
  if (binaryExtensions.test(path) || buffer.subarray(0, 8192).includes(0)) return null;
  return decoder.decode(buffer);
}

function scanBuffer(path, buffer) {
  const normalized = normalizePath(path);
  const findings = [...scanFilename(normalized, labelPath, markers)];
  const text = decodeText(buffer, normalized);
  if (text !== null) {
    findings.push(...scanText(normalized, text, labelPath));
    findings.push(...scanLocalMarkers(normalized, text, markers, labelPath));
  }
  return findings;
}

function uniqueFindings(findings) {
  return [...new Map(findings.map((item) => [
    [item.ruleId, item.path, item.line ?? ""].join("|"),
    item,
  ])).values()];
}

function finish(findings) {
  const remaining = uniqueFindings(applyAllowlist(findings, allowlist));
  if (remaining.length === 0) return;
  process.stdout.write(formatFindings(remaining) + "\n");
  process.exitCode = 2;
}

function nulList(args) {
  return decoder.decode(git(args, true)).split("\0").filter(Boolean);
}

function scanWorktree() {
  const normalCandidates = nulList(["ls-files", "-z", "--cached", "--others", "--exclude-standard"]);
  const ignoredCandidates = nulList(["ls-files", "-z", "--others", "--ignored", "--exclude-standard"]);
  const findings = [];
  for (const path of new Set(normalCandidates)) {
    const normalized = normalizePath(path);
    if (excludedPath.test(normalized)) continue;
    const absolute = resolve(root, normalized);
    if (!existsSync(absolute)) continue;
    const stat = lstatSync(absolute);
    if (stat.isSymbolicLink()) {
      findings.push({ ruleId: "symlink-candidate", path: labelPath(normalized), line: null });
    } else if (!stat.isFile()) {
      throw new Error("Unsupported worktree entry");
    } else {
      findings.push(...scanBuffer(normalized, readFileSync(absolute)));
    }
  }
  for (const path of ignoredCandidates) {
    const normalized = normalizePath(path);
    if (excludedPath.test(normalized)) continue;
    findings.push(...scanFilename(normalized, labelPath, markers));
  }
  return findings;
}

function scanFile(inputPath) {
  const absolute = resolve(inputPath);
  const stat = lstatSync(absolute);
  if (stat.isSymbolicLink() || !stat.isFile()) throw new Error("Outbound input is not a regular file");
  const path = basename(absolute) || "outbound-file";
  const externalLabel = () => "[redacted-path:1]";
  const buffer = readFileSync(absolute);
  const findings = [...scanFilename(path, externalLabel, markers)];
  const text = decodeText(buffer, path);
  if (text !== null) {
    findings.push(...scanText(path, text, externalLabel));
    findings.push(...scanLocalMarkers(path, text, markers, externalLabel));
  }
  return findings;
}

function scanStaged() {
  const findings = [];
  for (const path of nulList(["diff", "--cached", "--name-only", "--diff-filter=ACMR", "-z"])) {
    const normalized = normalizePath(path);
    const index = git(["ls-files", "--stage", "--", normalized]).trim();
    if (!index) throw new Error("Staged entry unavailable");
    const mode = index.split(/\s+/, 1)[0];
    if (mode === "120000") {
      findings.push({ ruleId: "symlink-candidate", path: labelPath(normalized), line: null });
    } else if (!mode.startsWith("100")) {
      throw new Error("Unsupported staged entry");
    } else {
      findings.push(...scanBuffer(normalized, git(["show", ":" + normalized], true)));
    }
  }
  return findings;
}

function validateRevision(revision) {
  if (revision === "--remotes=origin") {
    if (!git(["for-each-ref", "--format=%(refname)", "refs/remotes/origin"]).trim()) {
      throw new Error("Origin refs unavailable");
    }
    return;
  }
  const endpoints = revision.split("..");
  if (endpoints.length > 2 || endpoints.some((value) => !revisionToken.test(value) || value.startsWith("-"))) {
    throw new Error("Unsupported revision");
  }
  for (const endpoint of endpoints) git(["rev-parse", "--verify", endpoint + "^{object}"]);
}

function scanGitIdentity(label, raw, kind, owners) {
  const match = new RegExp("^" + kind + " (.*) <([^<>]*)> \\d+ [+-]\\d{4}$", "m").exec(raw);
  if (!match) throw new Error("Malformed Git identity");
  return scanIdentity(label, match[1], match[2], owners, markers, labelPath);
}

function scanTagObject(object, owners) {
  if (git(["cat-file", "-t", object]).trim() !== "tag") return [];
  const raw = decoder.decode(git(["cat-file", "tag", object], true));
  const boundary = raw.indexOf("\n\n");
  if (boundary < 0) throw new Error("Malformed tag object");
  const findings = [
    ...scanGitIdentity("tag-object", raw.slice(0, boundary), "tagger", owners),
    ...scanText("tag-message", raw.slice(boundary + 2), labelPath),
    ...scanLocalMarkers("tag-message", raw.slice(boundary + 2), markers, labelPath),
  ];
  return findings;
}

function scanRevision(revision) {
  validateRevision(revision);
  const owners = ownerHandles();
  const findings = [];
  const pathsByBlob = new Map();
  const commits = git(["rev-list", revision]).split("\n").filter(Boolean);
  for (const commit of commits) {
    const raw = decoder.decode(git(["cat-file", "commit", commit], true));
    const boundary = raw.indexOf("\n\n");
    if (boundary < 0) throw new Error("Malformed commit object");
    const label = "commit:" + commit.slice(0, 12);
    findings.push(...scanGitIdentity(label, raw.slice(0, boundary), "author", owners));
    findings.push(...scanGitIdentity(label, raw.slice(0, boundary), "committer", owners));
    findings.push(...scanText(label, raw.slice(boundary + 2), labelPath));
    findings.push(...scanLocalMarkers(label, raw.slice(boundary + 2), markers, labelPath));
    for (const row of decoder.decode(git(["ls-tree", "-r", "-z", commit], true)).split("\0").filter(Boolean)) {
      const tab = row.indexOf("\t");
      if (tab < 0) throw new Error("Malformed tree entry");
      const [mode, type, object] = row.slice(0, tab).split(" ");
      const path = normalizePath(row.slice(tab + 1));
      findings.push(...scanFilename(path, labelPath, markers));
      if (mode === "120000") findings.push({ ruleId: "symlink-candidate", path: labelPath(path), line: null });
      else if (type !== "blob") findings.push({ ruleId: "unsupported-git-entry", path: labelPath(path), line: null });
      else {
        const paths = pathsByBlob.get(object) ?? new Set();
        paths.add(path);
        pathsByBlob.set(object, paths);
      }
    }
  }
  for (const [object, paths] of pathsByBlob) {
    const buffer = git(["cat-file", "blob", object], true);
    for (const path of paths) findings.push(...scanBuffer(path, buffer));
  }
  return findings;
}

function scanHistory(revision) {
  const owners = ownerHandles();
  const findings = scanRevision(revision);
  const tagRows = git(["for-each-ref", "--format=%(refname)%09%(objectname)", "refs/tags"])
    .split("\n")
    .filter(Boolean);
  for (const row of tagRows) {
    const tab = row.indexOf("\t");
    if (tab < 0) throw new Error("Malformed tag ref");
    const ref = row.slice(0, tab);
    const object = row.slice(tab + 1);
    findings.push(...scanText("tag-ref", ref, () => "tag-ref"));
    findings.push(...scanLocalMarkers("tag-ref", ref, markers, () => "tag-ref"));
    findings.push(...scanTagObject(object, owners));
  }
  return findings;
}

function scanPush(input) {
  const findings = [];
  const rows = input.trim().split(/\r?\n/).filter(Boolean);
  if (rows.length === 0) throw new Error("Missing pre-push input");
  const owners = ownerHandles();
  for (const row of rows) {
    const fields = row.trim().split(/\s+/);
    if (fields.length !== 4) throw new Error("Malformed pre-push input");
    const [localRef, localObject, remoteRef, remoteObject] = fields;
    if (zeroObject.test(localObject)) continue;
    findings.push(...scanText("ref-name", localRef + "\n" + remoteRef, () => "ref-name"));
    findings.push(...scanLocalMarkers("ref-name", localRef + "\n" + remoteRef, markers, () => "ref-name"));
    findings.push(...scanTagObject(localObject, owners));
    findings.push(...scanRevision(zeroObject.test(remoteObject) ? localObject : remoteObject + ".." + localObject));
  }
  return findings;
}

function validateArchiveEntry(entry) {
  if (!entry || entry.startsWith("/") || entry.includes("\\") || entry.split("/").includes("..")) {
    throw new Error("Unsafe archive entry");
  }
}

function scanPackage(inputPath) {
  const archive = resolve(inputPath);
  const stat = lstatSync(archive);
  if (stat.isSymbolicLink() || !stat.isFile() || !archive.endsWith(".tgz")) {
    throw new Error("Invalid package archive");
  }
  const entries = decoder.decode(execFileSync("tar", ["-tzf", archive], {
    encoding: null,
    stdio: ["ignore", "pipe", "pipe"],
  })).split("\n").filter(Boolean);
  const details = decoder.decode(execFileSync("tar", ["-tvzf", archive], {
    encoding: null,
    stdio: ["ignore", "pipe", "pipe"],
  })).split("\n").filter(Boolean);
  if (entries.length === 0 || entries.length !== details.length) throw new Error("Malformed package archive");
  entries.forEach(validateArchiveEntry);
  if (details.some((line) => !["-", "d"].includes(line[0]))) throw new Error("Unsupported archive entry");

  const temporaryRoot = mkdtempSync(join(tmpdir(), "comins-package-security-"));
  try {
    execFileSync("tar", ["-xzf", archive, "-C", temporaryRoot], { stdio: ["ignore", "pipe", "pipe"] });
    const walk = (directory) => readdirSync(directory, { withFileTypes: true }).flatMap((entry) => {
      const absolute = join(directory, entry.name);
      const path = relative(temporaryRoot, absolute).split(sep).join("/");
      if (entry.isSymbolicLink()) return [{ ruleId: "symlink-candidate", path: labelPath(path), line: null }];
      if (entry.isDirectory()) return walk(absolute);
      if (!entry.isFile()) return [{ ruleId: "unsupported-archive-entry", path: labelPath(path), line: null }];
      return scanBuffer(path, readFileSync(absolute));
    });
    return walk(temporaryRoot);
  } finally {
    rmSync(temporaryRoot, { recursive: true, force: true });
  }
}

try {
  const [mode, argument, extra] = process.argv.slice(2);
  if (extra) throw new Error("Unexpected argument");
  if (mode === "worktree" && !argument) finish(scanWorktree());
  else if (mode === "file" && argument) finish(scanFile(argument));
  else if (mode === "staged" && !argument) finish(scanStaged());
  else if (mode === "history") finish(scanHistory(argument ?? "HEAD"));
  else if (mode === "push" && !argument) finish(scanPush(readFileSync(0, "utf8")));
  else if (mode === "package" && argument) finish(scanPackage(argument));
  else throw new Error("Invalid invocation");
} catch {
  process.stderr.write("security-audit-error: scan-failed\n");
  process.exitCode = 1;
}

- [ ] **Step 3: Implement the checksum-pinned installer**

Create `scripts/install-gitleaks.mjs`:

```js
import { execFileSync } from "node:child_process";
import { createHash } from "node:crypto";
import {
  chmodSync,
  mkdirSync,
  mkdtempSync,
  renameSync,
  rmSync,
  writeFileSync,
} from "node:fs";
import { tmpdir } from "node:os";
import { dirname, join, resolve } from "node:path";
import { pathToFileURL } from "node:url";

const version = "8.30.1";
const assets = new Map([
  ["linux-x64", {
    file: "gitleaks_8.30.1_linux_x64.tar.gz",
    sha256: "551f6fc83ea457d62a0d98237cbad105af8d557003051f41f3e7ca7b3f2470eb",
  }],
  ["darwin-arm64", {
    file: "gitleaks_8.30.1_darwin_arm64.tar.gz",
    sha256: "b40ab0ae55c505963e365f271a8d3846efbc170aa17f2607f13df610a9aeb6a5",
  }],
]);

export function assetFor(platform, arch) {
  const asset = assets.get(platform + "-" + arch);
  if (!asset) throw new Error("Unsupported Gitleaks platform");
  return asset;
}

export function verifyDigest(bytes, expected) {
  const actual = createHash("sha256").update(bytes).digest("hex");
  if (actual !== expected) throw new Error("Gitleaks checksum mismatch");
}

async function install(outputPath) {
  const asset = assetFor(process.platform, process.arch);
  const url = "https://github.com/gitleaks/gitleaks/releases/download/v" + version + "/" + asset.file;
  const response = await fetch(url, { redirect: "follow" });
  if (!response.ok) throw new Error("Gitleaks download failed");
  const bytes = Buffer.from(await response.arrayBuffer());
  verifyDigest(bytes, asset.sha256);
  const temporaryRoot = mkdtempSync(join(tmpdir(), "comins-gitleaks-"));
  try {
    const archive = join(temporaryRoot, asset.file);
    writeFileSync(archive, bytes, { mode: 0o600 });
    execFileSync("tar", ["-xzf", archive, "-C", temporaryRoot, "gitleaks"], {
      stdio: ["ignore", "pipe", "pipe"],
    });
    const target = resolve(outputPath);
    mkdirSync(dirname(target), { recursive: true });
    rmSync(target, { force: true });
    renameSync(join(temporaryRoot, "gitleaks"), target);
    chmodSync(target, 0o755);
    process.stdout.write(target + "\n");
  } finally {
    rmSync(temporaryRoot, { recursive: true, force: true });
  }
}

const invoked = process.argv[1] ? pathToFileURL(resolve(process.argv[1])).href : "";
if (import.meta.url === invoked) {
  const args = process.argv.slice(2);
  if (args.length !== 2 || args[0] !== "--output" || !args[1]) {
    throw new Error("Usage: install-gitleaks.mjs --output <path>");
  }
  await install(args[1]);
}
```

The CLI requires exactly `--output <path>`, verifies the official per-asset SHA-256 before extraction, writes temporary archives with mode `0600`, writes the executable with mode `0755`, and never places the binary in Git tracking.
