# Comins Contract v1.1 Adoption Completion Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Complete Contract v1.1 adoption across the independent Comins repositories and prepare secure post-bootstrap npm staged publishing without performing remote writes.

**Architecture:** `comins-governance` remains the shared policy source. Each package implements the policy with package-local documentation, CI, release workflow, verification, and commits; no workspace or shared runtime is introduced.

**Tech Stack:** Markdown, npm, Node.js 24, npm 11.15.0, GitHub Actions, OIDC trusted publishing, Vitest, Playwright.

## Global Constraints

- Keep `governance`, `data-table`, `grid-layout`, and `sortable` as independent Git repositories and commit units.
- Do not push, publish, tag, create a GitHub Release, mutate GitHub settings, or create npm packages.
- The first registry publication is interactive with maintainer 2FA because brand-new packages cannot use staged or trusted publishing.
- Post-bootstrap workflows use GitHub-hosted runners, Node 24, npm 11.15.0, the `npm` environment, `id-token: write`, and `npm stage publish`.
- Support only current stable Chromium-based Chrome and Edge until Firefox and WebKit projects are added.
- Preserve historical KMSF references in reports while keeping active instructions standalone.

- [ ] **Step 2: Add the guarded publish workflow**

Create `.github/workflows/publish.yml` with this control flow:

```yaml
name: Stage npm package

on:
  workflow_dispatch:
    inputs:
      version:
        description: Exact package.json version to stage
        required: true
        type: string

permissions:
  contents: read

concurrency:
  group: npm-stage
  cancel-in-progress: false

jobs:
  stage:
    if: github.ref == 'refs/heads/main'
    environment: npm
    runs-on: ubuntu-latest
    permissions:
      contents: read
      id-token: write
    steps:
      - uses: actions/checkout@v6
      - uses: actions/setup-node@v6
        with:
          node-version: '24'
          registry-url: https://registry.npmjs.org
          package-manager-cache: false
      - run: npm install --global npm@11.15.0
      - name: Verify requested version
        env:
          REQUESTED_VERSION: ${{ inputs.version }}
        run: node -e 'const { version } = require("./package.json"); if (version !== process.env.REQUESTED_VERSION) { console.error(`Expected ${version}, received ${process.env.REQUESTED_VERSION}`); process.exit(1); }'
      - name: Require bootstrap package
        run: node -e 'const { execFileSync } = require("node:child_process"); const { name } = require("./package.json"); execFileSync("npm", ["view", name, "version"], { stdio: "inherit" });'
      - run: npm ci
      - run: npm run verify
      - run: npm run test:consumer
      - run: npm pack --dry-run --json
      - run: npm stage publish --access public
```

- [ ] **Step 1: Add the consumer smoke gate**

Add `"test:consumer": "node scripts/consumer-smoke.mjs"` to package scripts and create this script:

```js
import assert from "node:assert/strict";
import { execFileSync } from "node:child_process";
import { mkdtemp, mkdir, rm, writeFile } from "node:fs/promises";
import { createRequire } from "node:module";
import { tmpdir } from "node:os";
import { dirname, join } from "node:path";
import { fileURLToPath, pathToFileURL } from "node:url";

const scriptsRoot = dirname(fileURLToPath(import.meta.url));
const repositoryRoot = dirname(scriptsRoot);
const temporaryRoot = await mkdtemp(join(tmpdir(), "comins-grid-layout-consumer-"));
const npmEnvironment = {
  ...process.env,
  npm_config_cache: join(temporaryRoot, "npm-cache"),
  npm_config_logs_dir: join(temporaryRoot, "npm-logs"),
};

await mkdir(npmEnvironment.npm_config_cache, { recursive: true });
await mkdir(npmEnvironment.npm_config_logs_dir, { recursive: true });

function runNpm(args, options = {}) {
  return execFileSync("npm", args, {
    cwd: repositoryRoot,
    encoding: "utf8",
    env: npmEnvironment,
    stdio: ["ignore", "pipe", "pipe"],
    ...options,
  });
}

try {
  runNpm(["run", "build"]);
  const packed = JSON.parse(runNpm(["pack", "--json", "--pack-destination", temporaryRoot]));
  const tarballName = packed[0]?.filename;
  assert.equal(typeof tarballName, "string", "npm pack did not return a tarball filename");

  const consumerRoot = join(temporaryRoot, "consumer");
  await mkdir(consumerRoot);
  await writeFile(
    join(consumerRoot, "package.json"),
    JSON.stringify({ name: "comins-grid-layout-consumer-smoke", private: true, type: "module" }),
  );

  execFileSync(
    "npm",
    ["install", "--ignore-scripts", "--no-audit", "--no-fund", "--no-package-lock", join(temporaryRoot, tarballName), "react@18", "react-dom@18"],
    { cwd: consumerRoot, env: npmEnvironment, stdio: "inherit" },
  );

  const requireFromConsumer = createRequire(join(consumerRoot, "package.json"));
  const resolved = requireFromConsumer.resolve("comins-grid-layout");
  await import(pathToFileURL(resolved).href);
  assert.ok(requireFromConsumer.resolve("comins-grid-layout/styles.css"));
  process.stdout.write("Consumer package smoke check passed.\n");
} finally {
  await rm(temporaryRoot, { force: true, recursive: true });
}
```
